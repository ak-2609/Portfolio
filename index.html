Projects

BikerX

The project focuses on cutting-edge technology designed to streamline bike servicing, featuring an Emergency SOS function, a smart chatbot, and calling services that address connectivity challenges. It also provides personalized recommendations for proactive care and includes API functionality. The application is developed using Android, Flutter, Dart, and Firebase, with Figma utilized for prototyping.

Pothole Detection System

A real-time pothole detection system using dashcam footage and Convolutional Neural Network (CNN) models to enhance road safety. Leveraging technologies such as Python, OpenCV, and TensorFlow/Keras, the system accurately detects and classifies potholes in various road conditions and lighting scenarios while continuously processing the dashcam footage.

HealthHub

The project is a web app that enhances healthcare access in rural areas through modular telemedicine kiosks, built with technologies like Node.js, Express.js, and MongoDB. Key features include Aadhaar API integration for data fetching, AI voice support, routing functionalities, Jitsi Meet for virtual consultations, and doctor prescription management. The kiosk also provides dashboard generation and email sending for comprehensive healthcare management.
